# 오버워치 폰트 파일

## 필요한 파일

이 디렉토리에 `koverwatch.ttf` 폰트 파일을 추가해주세요.

## 폰트 다운로드

koverwatch 폰트는 다음 위치에서 다운로드할 수 있습니다:
- https://www.dafont.com/koverwatch.font
- 또는 기타 무료 폰트 제공 사이트

## 설치 방법

1. `koverwatch.ttf` 파일을 다운로드합니다
2. 이 디렉토리(`styles/fonts/`)에 파일을 복사합니다
3. 파일명이 정확히 `koverwatch.ttf`인지 확인합니다

## 라이선스

koverwatch 폰트는 개인 및 상업적 용도로 무료로 사용할 수 있습니다.
자세한 라이선스 정보는 폰트 다운로드 페이지를 참조하세요.

**중요**: 폰트 파일은 저작권이 있는 자료이므로, 사용 전 라이선스를 확인하시기 바랍니다.
